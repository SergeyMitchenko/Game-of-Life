Game of Live

by Sergey Mitchenko

Instructions:

In the Menu on the right there are 3 Folders:

Visual Controller:

Change grid size (resolution X and resolution Y) or change the color of the live cells

Functional Controller:

Speed: Adjust the frame rate of the game

Presets: There are some interesting stored presets to experiment with

Calculation Algorithm: Choose the algorithm which is used to calculate the next generation of the Game of Live

Draw: Check to draw alive cells in the grid via mouse drag

Start (starts the game), Stop (stops the game), Next Generation (only calculates one generation further), Reset Grid (resets the Grid), Save (saves current grid into a cell.json file), Load (loads a chosen json file)

Information:

Shows general Information about the current game (Generation Number, Number of Alive Cells, Number of Births and Number of Deaths occurred in this generation)

Also shows the time the algorithm needed to calculate this generation
